  • Global vs Local
    • It's possible to have local and global variables with the same name.
    • The local variable takes precedence
    • Ej:
      var outerWear = "T-Shirt";

      function myOutfit() {
      var outerWear = "sweater";

      return outerWear;
      }

      console.log(myOutfit());
      Will return the line "sweater"

      But writing console.log("outerWear");
      still gives us "T-Shirt" (the global variable).
